## 1. 基于POSIX的共享内存实现的简单学生结构体的读写
```
./open # 创建1个名为"/abc"的共享内存,并输出状态
#-----------------------------------------------------------
# size=36
# mode=644
#-----------------------------------------------------------

# ./delete # 删除共享内存


./stat # 查看共享内存状态
#-----------------------------------------------------------
# size=36
# mode=644
#-----------------------------------------------------------


./send # 写入学生结构体的变量

./recv # 读取学生结构体的变量,并输出
#-----------------------------------------------------------
# name=orris age=15
#-----------------------------------------------------------

```
