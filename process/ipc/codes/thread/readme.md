## 1. POSIX线程的相关函数wait,attr,exit,key的使用
```
./attr # 设置创建的新线程的分离属性
#-----------------------------------------------------------------
# PTHREAD_CREATE_JOINABLE
# ABAB
#-----------------------------------------------------------------


./wait # 主线程等待新线程执行完毕后才结束
#-----------------------------------------------------------------
# ABABBBBBBBBBBBBBBBBBBB
#-----------------------------------------------------------------

./exit # 可能是测试线程的退出值把...

./key # TSD的主线程创建和新线程内使用
#-----------------------------------------------------------------
# Creating key....
# threadid=0x2209b700 is setting key
# threadid=0x2209b700 name=orris age=1
# threadid=0x2189a700 is setting key
# threadid=0x2189a700 name=orris age=0
# threadid=0x2189a700 name=orris age=0
# destroy...
# threadid=0x2209b700 name=orris age=1
# destroy...
#-----------------------------------------------------------------

```
