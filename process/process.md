## 1. 进程共享
父子进程fork之后:
1. 相同之处
    1. 用户空间(其实全部都复制一遍)
        1. 全局变量
        2. .data
        3. .text
        4. 栈
        5. 堆
        6. 环境变量
    2. 用户id
    3. 宿主目录(家目录)
    4. 进程工作目录
    5. 信号处理方式
    6. 文件描述符
    7. mmap建立的映射区
2. 不同之处
    1. 进程id
    2. fork返回值
    3. 父进程id
    4. 进程运行时间
        + 父进程从main开始,而子进程从fork开始
    5. 闹钟(定时器)
        + 1个进程有1个闹钟
    6. 未决信号集
