## 1. epoll的水平触发和边沿触发基于管道的比较实现
1. 原理
    1. 写端不断写入10个数据,而读端每次只读5个数据并输出.分别用水平触发和边沿触发来测试
2. 现象
    1. 水平触发:读端一下子就输出2行.(其实是循环了2次)
    2. 边沿触发:读端只能输出1行,过一会儿才能输出第2行=>管道里的剩余5个数据留到下次才读
```
./et 
#----------------------------------------------------
# aaaa
# bbbb
# cccc
#----------------------------------------------------

```
